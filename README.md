# UniTest

UniTest is a web-based tool that modifies numerical (classical) questions while keeping their logical structure intact. It uses Vision-LLMs to parse mathematical text from exam photos and regenerates them with new numerical values.

## Features
- **One-Shot Prompting:** Custom logic for GPT-4o to swap numerical values in math problems without breaking constraints.
- **Vision OCR:** GPT-4o-mini Vision integration to extract and structure data from exam images.
- **Backend:** A lightweight Python (Flask) API with SQLite for data storage.

## Tech Stack
- **Languages:** Python, JavaScript, HTML, CSS
- **Framework:** Flask
- **AI:** OpenAI API (GPT-4o & GPT-4o-mini Vision)
- **Database:** SQLite
